# DNA Future Lab

אתר לימודי עתידני בעברית על מנגנון ה־DNA.

## מה כבר כלול

- עמוד בית עתידני ורספונסיבי
- תפריט המבורגר צדדי מימין עם גלילה פנימית
- תמיכה מלאה ב־RTL
- שישה עמודי לימוד נפרדים
- אנימציית DNA שנבנתה ב־CSS ללא ספריות חיצוניות
- כפתורי מעבר בין הפרקים
- התאמה למחשב, לטאבלט ולטלפון

## הפעלה מקומית

פתחו את `index.html` בדפדפן, או הריצו שרת מקומי פשוט.

## פרסום ב־GitHub Pages

במאגר: Settings → Pages → Deploy from a branch → בחרו `main` ואת התיקייה `/ (root)`.
